# 안녕하세요, 첫 글입니다

이 블로그는 **마크다운** 파일을 읽어 정적 HTML로 변환하는 방식으로 만들어졌습니다. 프레임워크 없이 순수한 HTML, CSS, JavaScript만 사용합니다.

## 지원하는 문법들

이 문단에는 *기울임*과 **굵게**, 그리고 `인라인 코드`가 섞여 있습니다. 자세한 내용은 [프로젝트 문서](https://example.com)를 참고하세요.

### 목록

순서 없는 목록:

- 마크다운 파싱
- 다크 모드 지원
- 모바일 반응형 레이아웃

순서 있는 목록:

1. posts 폴더에 글 작성
2. 빌드 스크립트 실행
3. dist 폴더 배포

### 코드 블록

```
function hello() {
  console.log("Hello, world!");
}
```

### 인용구

> 읽기 좋은 글이 좋은 블로그를 만든다.
